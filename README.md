# PinReady

> **Projet non poursuivi** — Ce projet etait un test/prototype de configurateur pour VPinballX standalone sous Linux. Je me lance desormais dans l'aventure [Batocera](https://batocera.org/) pour mon pincab.

PinReady est un configurateur et lanceur cross-platform pour VPinballX (Visual Pinball X standalone 10.8.1).
Il gere l'assignation des ecrans, le mapping des inputs, la sensibilite tilt/nudge, le routage audio, et le lancement de tables.

Stack : Rust, eframe/egui, SDL3, SQLite.
